# 音の種類別設定（基本編）

子音の種類によって、原音設定の方法が変わってきます。
このチュートリアルでは、よくある子音の種類ごとに設定のポイントを説明していきます。

> **注意:** このチュートリアルの説明は単独音を前提としています。連続音を設定する場合は、各説明の**先行発声の部分のみ**を参考にしてください。

> は行、や行、わ行については、サンプルが多いため[チュートリアル14](/docs/tutorial-ja-14-音の種類別設定-応用編--docs)で説明します。

## 摩擦音の場合

摩擦音の子音は、高い周波数が強く出るのが特徴です（スペクトログラムの上の方の色が濃くなります）。
子音が比較的長くなることが多いです。

**このパターンで設定する音:**
- さ行、ざ行
- しゃ行、じゃ行
- ひゃ行

### ざ行の例

<iframe
  src="/iframe.html?id=components-editor-canvasbase--fricative-z&viewMode=story"
  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="摩擦音：ざ"
/>

単独音「ざ」の設定例です。

**設定のポイント:**

- **左ブランク**: 子音の最初が再生されないと違う音に聞こえてしまうことがあるため、必ず子音全体を含めるようにします。子音が長すぎると滑舌が悪く聞こえますが、子音の頭が切れないようにすることを優先します。

- **オーバーラップ**: 先行発声の1/3程度に設定します。オーバーラップは、単独音では音の立ち上がりの時間になります。オーバーラップを大きくしすぎると、子音の頭がよく聞こえなくなって違う音に聞こえてしまうことがあるため、上げすぎないようにします。

- **先行発声**: 低い周波数が鳴り出す部分（スペクトログラムの下の方に濃い色が出てくる部分）から母音が始まりますので、ここに先行発声を合わせます。

## 鼻音の場合

鼻音の子音は、低い周波数に弱めの音が出て、高い周波数はほとんど出ないのが特徴です（スペクトログラムの下の方が薄い色、上の方は無色になりがちです）。

**このパターンで設定する音:**
- な行、ま行
- にゃ行、みゃ行

  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="鼻音：ま"
/>

単独音「ま」の設定例です。

### な行の例

<iframe
  src="/iframe.html?id=components-editor-canvasbase--nasal-n&viewMode=story"
  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="鼻音：な"
/>

単独音「な」の設定例です。

**設定のポイント:**

- **左ブランク**: 基本的には子音全体が入るようにしますが、鼻音は子音の頭が少し切れても同じ音に聞こえるため、子音が長すぎる場合は切ることができます。

- **オーバーラップ**: 先行発声の1/3〜1/2程度に設定します。少し長めに設定したほうが滑らかに聞こえます。

- **先行発声**: 母音が始まってくると、スペクトログラムの中域にも濃い色が現れてきますので、ここに設定します。

## 無声破裂音の場合

破裂音は、低い音から高い音まで満遍なく強い部分（スペクトログラム全体が濃い色）と、その**直前の無音部分**でできています。
音が鳴る前の溜めの部分をしっかり原音設定に反映することが大切です。

**このパターンで設定する音:**
- か行、きゃ行
- た行、てゃ行
- ぱ行、ぴゃ行
- ちゃ行、つぁ行

> **注意:** ちゃ行とつぁ行は、無音部分のあとに摩擦音のようなスペクトログラムが現れますが（破擦音）、考え方は同じで無音部分を表す設定にしていきます。

### か行の例

<iframe
  src="/iframe.html?id=components-editor-canvasbase--plosive-k&viewMode=story"
  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="無声破裂音：か"
/>

単独音「か」の設定例です。

**設定のポイント:**

- **左ブランク**: 音が鳴り始める部分に設定します。

- **オーバーラップ**: -20〜-50程度に設定します。このマイナスのオーバーラップが溜めの部分を作ります。

- **先行発声**: 母音が始まってくると、スペクトログラムの高域の音が弱くなりますので、ここに設定します。

## 有声破裂音の場合

**このパターンで設定する音:**
- が行、ぎゃ行
- だ行、でゃ行
- ば行、びゃ行
- ら行、りゃ行

> **注意:** が行が鼻音化している場合は、鼻音と同じような設定にします。

## 母音の場合

母音は、低域〜中域に濃い色が出るのが特徴です。
単独音の場合と連続音の場合で設定方法が変わってきます。

**このパターンで設定する音:**
- あ行（あ、い、う、え、お）
- ん

### 単独音の場合

<iframe
  src="/iframe.html?id=components-editor-canvasbase--blank-range-stable&viewMode=story"
  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="単独音の母音：あ"
/>

単独音「あ」の設定例です。

**設定のポイント:**

- **左ブランク**: 音が鳴り始める部分に設定します。

- **オーバーラップ**: 0に設定します。

- **先行発声**: 0に設定します。

### 連続音：同じ母音が続く場合

<iframe
  src="/iframe.html?id=components-editor-canvasbase--vowel-a-a&viewMode=story"
  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="母音連続音：a あ"
/>

連続音「a あ」の設定例です。同じ母音が続く場合の設定です。

正しく録音している場合、「ああ」は「あー」と発音しているはずです（「あ、あ」と区切って発音している場合は録音しなおした方がよいです）。
この2拍分の「あ」の範囲にすべてのパラメータが収まっていれば、あまり設定に拘らなくても大丈夫です。

### 連続音：異なる母音が続く場合

<iframe
  src="/iframe.html?id=components-editor-canvasbase--vowel-a-i&viewMode=story"
  style={{ width: '100%', height: '250px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="母音連続音：a い"
/>

連続音「a い」の設定例です。異なる母音が続く場合の設定です。

**設定のポイント:**

母音部分は、いずれも低域〜中域に濃い色が出ることに加えて、前の音から次の音に滑らかに遷移していくため、設定がやや難しいです。

スペクトログラムが **[前の母音の部分] - [遷移部分] - [次の母音の部分]** のようになっていることをまず見つけます。
先行発声はこの遷移部分のどこかに置き、再生ボタンも駆使しながら良いタイミングを見つけます。

上の例では、遷移部分が終わって明らかに次の母音になったところに設定しています。
