# 付録03. スマートフォンで録音する

このチュートリアルでは、スマートフォンでUTAU音源を録音する方法を説明します。
UTAU音源向け録音アプリ**RecStar**を使用して、Hello, UTAU World! キットで録音を進めます。

> 💡 **注意**: この付録は、パソコンを持っていない方がスマートフォンだけで音源制作を完結させることを想定しています。

## 準備（物理的な環境）

録音を始める前に、以下のものを用意してください。

### 有線イヤフォンまたはヘッドフォン

スマートフォンに接続できる**有線**のイヤフォンかヘッドフォンを用意してください。

**なぜ有線が必要？**
- ガイドBGMを再生しながら録音します
- ガイドBGMがスマホのマイクに入らないようにする必要があります
- 音の遅延があると録音がうまくいきません

> ⚠️ **注意**: 無線イヤフォン（Bluetooth）は音の遅延があるため、録音には適していません。

> 💡 **ヒント**: イヤフォンにマイク機能が付いている場合、スマートフォンが誤ってイヤフォンのマイクで録音してしまうことがあります。その場合、音質が悪くなったり、ガイドBGMが混入したりする可能性があります。可能であれば、マイク機能のない音楽再生専用のイヤフォンやヘッドフォンを使用することを推奨します。

### 声を出しても大丈夫な環境

録音では、「あーかーきーくーけーこー」「いーさーしーすーせーそー」といった声を、元気よく繰り返し発声します。

**必要なこと:**
- 家族や周囲の人に事前に説明しておく
- 気兼ねなく声を出せる時間帯を選ぶ
- 可能であれば、録音することを理解してもらう

### （できれば）雑音が少ない環境

練習用の音源であれば、それほど神経質になる必要はありませんが、以下の音は録音の邪魔になります。

**避けたい音:**
- 電車や車の音
- 虫の鳴き声
- エアコンやファンの音
- テレビや話し声

静かな時間帯や場所を選ぶとよいでしょう。

### （できれば）反響が少ない環境

音声合成では、反響音（エコー）は品質を下げる原因になります。

**よい環境:**
- 布団やカーテンなど、布製品に向かって録音する
- 布や毛布で囲んだ空間

**避けたい環境:**
- お風呂場やトイレ（反響が強い）
- 広い部屋の中央
- 壁に向かって直接録音

## 準備（アプリとファイルのダウンロード）

以下のアプリとファイルをダウンロードします。

### RecStarアプリ

お使いのスマートフォンに合わせて、以下からダウンロードしてください。

- [RecStar (Android)](https://play.google.com/store/apps/details?id=com.sdercolin.recstar&hl=ja)
- [RecStar (iOS)](https://apps.apple.com/jp/app/recstar/id6471727990)

### Hello, UTAU World! キット

- [Hello, UTAU World!](https://bowlroll.net/file/318744)

このキットには、録音リスト・ガイドBGM・原音設定用テンプレートがすべて含まれています。

## Hello, UTAU World! のzipを解凍する

ダウンロードしたzipファイルを解凍します。

**Android の場合:**

1. ダウンロードしたzipファイルをタップ
2. 「展開」または「解凍」を選択
3. 解凍先のフォルダを選択（わからない場合は「ダウンロード」フォルダでOK）

**iOS の場合:**

1. ダウンロードしたzipファイルをタップ
2. 自動的に解凍されます
3. 「ファイル」アプリで確認できます

解凍すると、以下のフォルダとファイルが表示されます：

```
Hello UTAU World!_v110/
  └ Hello UTAU World!/
      ├ ガイドBGM/
      ├ 録音リスト/
      ├ otoiniテンプレート/
      ├ character txt テンプレート/
      ├ character yaml テンプレート/
      ├ voice/
      ├ テスト歌唱用データ/
      └ readme（説明・利用規約など）.txt
```

## ガイドBGMを選ぶ

`ガイドBGM` フォルダの中には、BPM120のガイドBGMが5種類入っています。

**種類:**

各種類ごとにフォルダがあり、その中に音程別のファイルが入っています。

- **Ethnic** - C、E、G#の音程
- **Pop** - A#、D、F#の音程
- **Rock** - A、C#、Fの音程
- **Soft** - C、E、G#の音程
- **Sweet** - A#、D、F#の音程

**選び方:**

1. それぞれのガイドBGMを再生してみる
2. 再生しながら実際に声を出してみる
3. 一番歌いやすい（声を出しやすい）ものを選ぶ

> 💡 **ヒント**: 自分の声の高さに合ったものを選びましょう。無理に高い音や低い音に合わせる必要はありません。

## RecStarで録音の準備をする

### 1. RecStarを起動する

アプリを開くと、セッション画面が表示されます。

<img src="/screenshot/01_recstar_session.png" alt="RecStarセッション画面" style={{ maxWidth: '400px', width: '100%', height: 'auto' }} />

### 2. 新しいセッションを作成する

画面右下の **[+]** ボタンをタップします。

### 3. 録音リストをインポートする

録音リストを選ぶ画面になります。

<img src="/screenshot/02_recstar_reclist_none.png" alt="RecStar録音リスト選択画面" style={{ maxWidth: '400px', width: '100%', height: 'auto' }} />

1. 画面右上の **[⋮]**をタップ
2. **[+ インポート]** を選択
3. ファイル選択画面になります

**ファイルの場所:**

先ほど解凍した `Hello UTAU World!_v110` フォルダの中を開いていきます：

```
Hello UTAU World!_v110/
  → Hello UTAU World!/
    → 録音リスト/
      → まいこ製 最軽量リスト（切り出し単独音）.txt
```

このファイルを選択します。

> 💡 **ヒント**: ファイル選択画面で迷ったら、「ダウンロード」フォルダから探してみてください。

### 4. コメントファイルをインポートする

録音リストを選択すると、「コメントファイルをインポートしますか？」と聞かれます。

1. **[はい]** をタップ
2. 同じフォルダ内の `まいこ製 最軽量リスト（切り出し単独音）-comment.txt` を選択

> ⚠️ **注意**: [はい]を押した後、ファイルを選ばずにキャンセルすると、録音リスト自体もキャンセルされてしまいます。その場合は、もう一度手順3からやり直してください。

> 💡 **ヒント**: コメントファイルは、録音リストによっては配布されていない場合もあります。その場合は[いいえ]を選んでください。

### 5. 録音リストを選択する

録音リスト一覧に **[まいこ製 最軽量リスト（切り出し単独音）]** が表示されるので、タップして選択します。

<img src="/screenshot/03_recstar_reclist_loaded.png" alt="RecStar録音リスト読み込み後" style={{ maxWidth: '400px', width: '100%', height: 'auto' }} />

### 6. ガイドBGMを設定する

録音画面が表示されます。

1. 画面右上の **[⋮]** をタップ
2. **[ガイドBGMを設定]** を選択
3. 先ほど選んだガイドBGMのファイルを選択

**ガイドBGMの場所:**

```
Hello UTAU World!_v110/
  → Hello UTAU World!/
    → ガイドBGM/
      → UTAU用ガイドBGM 120 （種類） ver - by まいこ/
        → （種類） （音程）.wav
```

**例:**
- Softの C を選んだ場合:
  - フォルダ: `UTAU用ガイドBGM 120 Soft ver - by まいこ`
  - ファイル: `Soft C.wav`
- Ethnicの E を選んだ場合:
  - フォルダ: `UTAU用ガイドBGM 120 Ethnic ver - by まいこ`
  - ファイル: `Ethnic E.wav`

### 7. 音量を確認する

イヤフォンから音漏れがないか確認します。

<img src="/screenshot/04_rrecstar_recording_beforeRecording.png" alt="RecStar録音画面（録音前）" style={{ maxWidth: '400px', width: '100%', height: 'auto' }} />

1. **赤い丸ボタン**（録音ボタン）をタップ
2. ガイドBGMが流れますが、**声は出さず**に聞くだけにします
3. ガイドBGMが終わったら、**赤い停止ボタン**をタップ
4. 録音された波形が画面下部に表示されます
5. 波形の左上にある**再生ボタン**をタップして、録音内容を聞きます

**確認ポイント:**

声を出していないのに、ガイドBGMが聞こえる場合は、イヤフォンから音漏れしています。
→ スマートフォンの音量を下げてください。

<img src="/screenshot/05_recstar_recording_afterRecording.png" alt="RecStar録音画面（録音後）" style={{ maxWidth: '400px', width: '100%', height: 'auto' }} />

> 💡 **ヒント**: 実際に録音するときは、波形の大きさにも注目しましょう。波形がいっぱいいっぱいに表示されている場合、声が大きすぎる可能性があります。その場合は、スマートフォンから少し離れて録音してください。

## 録音する

準備ができたら、実際に録音を始めます。

### 録音の手順

1. **赤い丸ボタン**をタップ
2. ガイドBGMが流れるので、リズムに合わせて声を出します
3. 録音が終わったら**停止ボタン**をタップ
4. 録音内容を確認します（必要に応じて）
5. 録音ボタンの右隣にある **[>]** ボタンをタップして次の行へ進みます
6. すべての録音が終わるまで、手順1〜5を繰り返します

**Hello, UTAU World! の場合:**

全部で**23行**録音します。録音時間は、やり直しを含めて**3〜10分程度**を想定しています。

> 💡 **ヒント**: 言い間違えた場合は、次に進まずに**もう一度録音ボタン**を押せば、何度でもやり直すことができます。

### 録音のコツ

- **マイクとの距離を一定に**: マイクとの距離で音質が大きく変わります。できればスマートフォンを置いて、一定の距離で録音できるようにしましょう
- **リズムを守る**: ガイドBGMのリズムに合わせて発声します
- **一定の声量**: すべての行で同じくらいの声量で録音します
- **はっきりと発音**: 「あ」「か」「さ」などの違いがわかるように発音します
- **焦らない**: 言い間違えたらやり直せばOKです

## 録音したファイルをzipで保存する

すべての録音が終わったら、ファイルをzipで保存します。

### 1. エクスポートを選ぶ

1. 画面右上の **[⋮]** をタップ
2. **[エクスポート]** を選択

### 2. 保存先フォルダを選ぶ

録音した音源を保存するフォルダを選びます。

**おすすめの保存先:**

- **Android**: 「ダウンロード」フォルダまたは「Documents」フォルダ
- **iOS**: 「ファイル」アプリ内の「このiPhone内」または「iCloud Drive」

> 💡 **ヒント**: わからない場合は、「ダウンロード」フォルダを選んでおけば、後で見つけやすいです。

フォルダを選ぶと、録音したファイルが保存されます。

### 3. zipファイルを作成する

保存されたフォルダを、zipファイルにします。

**Android の場合:**

1. ファイル管理アプリ（「ファイル」や「マイファイル」など）を開く
2. 保存したフォルダを長押し
3. メニューから「圧縮」または「Zip」を選択
4. zipファイルが作成されます

**iOS の場合:**

1. 「ファイル」アプリを開く
2. 保存したフォルダを長押し
3. メニューから「圧縮」を選択
4. zipファイルが作成されます

> 💡 **ヒント**: zipファイルの名前は、わかりやすい名前に変更しておくとよいでしょう（例: `私の音源_20250107.zip`）。

## 次のステップ：LABERUで原音設定

録音が終わったら、LABERUで原音設定を進めます。

1. LABERUを開く
2. 作成したzipファイルを読み込む（[04.音源を読み込む](/docs/tutorial-ja-04-音源を読み込む--docs)参照）
3. Hello, UTAU World! のテンプレートを使用して原音設定（[06-1.外部からoto.iniを読み込む](/docs/tutorial-ja-06-1-外部からotoiniを読み込む--docs)参照）

**テンプレートの場所:**

```
Hello UTAU World!_v110/
  → Hello UTAU World!/
    → otoiniテンプレート/
      → BPM120用/
        → oto.ini
```

RecStarで録音したガイドBGMのBPMに合わせて、テンプレートを選びます。

- ガイドBGMが120の場合（通常はこれ）: `BPM120用/oto.ini`
- ガイドBGMが100の場合: `BPM100用/oto.ini`

このファイルをテンプレートとして読み込みます。

---

これで、スマートフォンだけで音源の録音から原音設定まで完結できます！
