## How to Use This Tutorial

### First-time Users

We recommend reading through the tutorial in order, starting from Chapter 01. Pay special attention to:

1. **Chapter 1** - Understand the basics of UTAU and oto configuration
2. **Chapter 2** - Load your voicebank and prepare it
3. **Chapter 3** - Actually configure the oto.ini parameters

### Creating Voicebanks with Only a Smartphone

If you don't have access to a PC, follow this order:

1. **Appendix 02** - Learn how to choose recording lists and guide BGM
2. **Appendix 03** - Learn how to record with your smartphone
3. **Chapters 1-3** - Learn how to configure oto.ini
4. **Chapter 4** - Learn how to use your voicebank with GAKUYA or utalet

### Working Efficiently on PC

After understanding the basics, make use of:

- **Appendix 01** - Memorize keyboard shortcuts for faster work
- **Chapter 11** - Use batch processing features
