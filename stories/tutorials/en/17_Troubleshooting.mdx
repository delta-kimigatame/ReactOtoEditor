# Troubleshooting

Here's how to handle problems you might encounter with this app.

## When Issues Occur

This app doesn't automatically notify the author when errors occur.
If you encounter bugs or problems, please send the **log file** to the author.

### How to Download Log File

1. Click the **menu button** (three lines icon) in the header
2. Select "**Show Log**"
3. Log dialog appears

<iframe
  src="iframe.html?id=components-showlogdialog-showlogdialog--opened&viewMode=story"
  style={{ width: '100%', height: '400px', border: '1px solid #ccc', pointerEvents: 'none' }}
  title="Log dialog"
/>

4. Tap the **"Download"** button
5. Log file (`log.txt`) downloads

### How to Send Log File

Send the downloaded log file to the author.

**Where to send:**

[Discord Server](https://discord.gg/aqZRj3CRcm)

**Tips for sending:**

- What operation caused the problem
- What error message appeared
- Your phone model and browser type

Providing this info together helps resolve issues faster.

> 💡 **Hint**: Screenshots make situations clearer.

## Common Problems and Solutions

### Problem 1: App Stopped Working

**Cause:**

Browser's auto-translate feature might be on.

**Solution:**

1. Open browser settings
2. Turn translation feature **off**
3. Reload page

> ⚠️ **Note**: This app won't work properly with browser auto-translate on. This is by design.

**How to set in major browsers:**

**Chrome (Android):**
1. Tap **︙** (3-dot menu) at top right
2. "Settings" → "Languages"
3. Turn off "Offer to translate pages"

**Safari (iPhone):**
1. Open "Settings" app on iPhone
2. "Safari" → "Advanced"
3. Turn off "Website Translation"

### Problem 2: Saved Content Disappeared

**Cause:**

- Browser cache was cleared
- Using a different browser

**Solution:**

We recommend taking regular backups.

See [15.Save and Download](/docs/tutorial-en-15-save-and-download--docs) for details.

### Problem 3: Can't Download

**Cause:**

- Downloads blocked in browser settings
- Insufficient storage space

**Solution:**

1. Check browser download settings
2. Check phone storage space
3. Delete unnecessary files to free space

### Problem 4: Can't Load Audio Files

**Cause:**

- Incorrect zip file format
- Audio file format not supported

**Solution:**

This app only supports **WAV files**.
Other formats (MP3, OGG, etc.) can't be loaded.

## If Still Not Resolved

If the above methods don't solve it, send your log file to the author.

[Discord Server](https://discord.gg/aqZRj3CRcm)

The author will provide direct support.
