# LABERU User Guide

Welcome to LABERU! This guide will walk you through how to configure oto.ini files for UTAU voicebanks step by step.

## What is LABERU?

LABERU is a web application that allows you to edit oto.ini files for UTAU voicebanks directly in your browser.
No installation required - you can access it from any device.

> 💡 **Want to learn more about UTAU and oto configuration?** Check out [What is UTAU?](/docs/tutorial-en-02-what-is-utau--docs)

## Features

- 📊 **Waveform and Spectrogram Display** - Visualize audio while editing
- ✏️ **Intuitive Editing** - Adjust parameters easily with mouse or touch controls
- 💾 **Easy Saving** - Download your edited oto.ini file instantly
- 🎨 **Customizable** - Personalize with theme colors, touch mode, and more

## What You'll Need

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- A UTAU voicebank to edit (zip file containing wav files)

## How to Use This Guide

Navigate through each topic using the menu.
Sample screenshots of the actual interface are included so you can follow along as you learn.
