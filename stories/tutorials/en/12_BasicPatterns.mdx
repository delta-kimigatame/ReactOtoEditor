# Basic Oto Patterns

UTAU oto settings follow set patterns depending on the sound type.
Here we'll introduce common patterns.

## CV (Single Sounds)

Format recording one sound per file (e.g., a.wav).

- **Left blank**: Varies by consonant type
- **Overlap**: Adjust to consonant type
- **Preutterance**: Moment when consonant changes to vowel
- **Fixed range**: Where vowel volume stabilizes
- **Right blank**: In stable vowel range, as far from fixed range as possible

## Vowel Connection CV

Special CV for smoothly connecting vowels (e.g., [* a]).
Six types: "a i u e o n".

- **Left blank**: Where vowel stabilizes (around normal CV fixed range)
- **Overlap**: 2x preutterance
- **Preutterance**: Around 25-50 (adjust to preference)
- **Fixed range**: Slightly larger than overlap
- **Right blank**: Set like normal CV

## VCV (Simple Settings)

If your recorded guide BGM and auto-generation settings match, it's easy to set.

- **Preutterance**: Only adjust the consonant-to-vowel transition moment

## VCV (Non-first Sounds)

Middle and last sounds in VCV connect from the previous sound.

- **Left blank**: Where previous vowel is stable
- **Overlap**: Where previous vowel is stable (1/3 of preutterance)
- **Preutterance**: Moment when consonant changes to vowel
- **Fixed range**: Slightly larger than overlap
- **Right blank**: Set like CV

> In VCV, each record's preutterance and overlap have the same value.
> So basically you don't need to change left blank and overlap.
> 
> However, if overlap hits the consonant instead of vowel, adjustment is needed.
> Turn on Overlap Lock and move left blank left until overlap moves to the vowel part.

## CVVC: CV Part

CVVC combines CV (single sounds) and VC (vowel+consonant).
Set the CV part like regular CV.

## CVVC: VC Part

Connection from vowel to consonant.

- **Left blank**: Where vowel is stable
- **Overlap**: Where vowel is stable (1/3 of preutterance)
- **Preutterance**: Vowel-to-consonant boundary (around CV left blank)
- **Fixed range**: Between right blank and preutterance
- **Right blank**: Just before consonant changes to next vowel (slightly left of CV preutterance)

## Stretch Range Settings

Stretch range is the portion from fixed range to right blank.
Consider two points when setting this range:

1. **Choose where volume is stable**
2. **Make it as long as possible**

Balancing both is ideal.
