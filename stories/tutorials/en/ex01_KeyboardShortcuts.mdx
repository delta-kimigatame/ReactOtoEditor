# Appendix 01. Keyboard Shortcuts

When editing on PC, keyboard shortcuts make work much more efficient.

> ⚠️ **Note**: This feature is PC-only. Not available on smartphones or tablets.

## Parameter Editing

In the waveform display area, hold these keys while moving the mouse to adjust parameters.

**`Q`** - Change offset (left blank)

**`W`** - Change preutterance

**`E`** - Change overlap

**`R`** - Change consonant (fixed range)

**`T`** - Change blank (right blank)

> 💡 **How to use**: Hold a key and move the mouse left/right to change the corresponding parameter.

For detailed parameter explanations, see [03.What is Oto Settings](/docs/tutorial-en-03-what-is-oto--docs).

## Playback Functions

**`1`** - Play left blank~preutterance
- Check sound entrance

**`2`** - Play preutterance~right blank
- Check sound sustain

**`3`** - Play with metronome
- Check rhythm

> 💡 **Hint**: Convenient for confirming if preutterance position is correct. See [09.Editor Screen Basics](/docs/tutorial-en-09-editor-screen-basics--docs) for details.

## Mode Switching

**`4`** - Toggle overlap lock
- Auto-adjust overlap to 1/3 of preutterance

**`5`** - Toggle tap mode
- Move preutterance to tapped position

> 💡 **Hint**: For VCV editing, turn both on for efficiency.

## Dialog Display

**`6`** - Show alias edit screen
- Detailed alias name and parameter editing

**`7`** - Show parameter table
- View all parameters / batch processing

For detailed usage:
- [10.Alias Edit Screen](/docs/tutorial-en-10-alias-edit-screen--docs)
- [11.Parameter Table and Batch Processing](/docs/tutorial-en-11-parameter-table-and-batch-processing--docs) 

### Zoom

**`+` (Plus)** - Zoom in
- Enlarge waveform display

**`-` (Minus)** - Zoom out
- Reduce waveform display

> ⚠️ **Note**: Pinch zoom is not supported.

## Alias Navigation

**`←` or `↑`** - Previous alias
- Show previous alias

**`→` or `↓`** - Next alias
- Show next alias

> 💡 **Hint**: Arrow keys let you quickly switch aliases.

## Efficient Editing Flow with Shortcuts

Once comfortable, you can work very fast with this flow:

1. **`W` key + mouse** to roughly adjust preutterance
2. **`1` key** to play left blank~preutterance and confirm
3. **`2` key** to play preutterance~right blank and confirm
4. Use **`Q`, `E`, `R`, `T` keys** as needed for other parameters
5. **`→` key** to move to next alias
6. Repeat steps 1-5

> 💡 **Tip**: Reducing mouse clicks greatly improves work speed.
