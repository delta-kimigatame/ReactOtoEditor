# What is UTAU?

Before using LABERU, let's briefly explain what UTAU is.

## What is UTAU?

**UTAU** is software that creates singing by combining recorded voice samples.
You prepare voice data (called a **voicebank**) containing sounds like "a," "ka," "sa," etc., then input melodies and lyrics - and your computer will sing!

What makes UTAU special is that **anyone can create their own voicebanks**.
This has led to **thousands upon thousands of voicebanks** being shared online - cute voices, cool voices, voices in many different languages, and more.

## What is a UTAU Voicebank?

A **UTAU voicebank** is a collection of voice data files used to make UTAU sing.

### What's Inside a Voicebank
- **wav files** - Recorded voice samples (like "a," "ka," etc.)
- **oto.ini** - Configuration file that tells UTAU how to use each sample (called **oto configuration**)
- Other configuration files

> 💡 **Want to learn more about oto configuration?** Check out [What is Oto.ini?](/docs/tutorial-en-03-what-is-oto--docs)

## Software That Uses UTAU Voicebanks

UTAU voicebanks can be used with various software:

#### <img src="icons/utau.png" alt="UTAU" width="30" /> UTAU (Original)
- Where it all began - the legendary software that started this culture
- For Windows only
- Can be challenging with newer voicebank types or non-Japanese environments
- [UTAU Official Site](https://utau2008.xrea.jp/)

#### <img src="icons/openutau.png" alt="OpenUtau" width="30" />  OpenUtau
- The current standard for PC-based UTAU
- Available for Windows, Mac, and Linux
- Handles various voicebank types smoothly
- Strong support for non-Japanese languages
- [OpenUtau Official Site](https://www.openutau.com/)

#### <img src="icons/utalet.png" alt="utalet" width="30" /> utalet
- Browser-based UTAU
- No installation needed - just open and use
- Works on smartphones and tablets too
- Great compatibility with LABERU (same developer!)
- [utalet Official Site](https://k-uta.jp/utalet/)
