# Sound Types (Basic)

Oto settings methods vary by consonant type.
This tutorial explains key points for each common consonant type.

> **Note:** This tutorial assumes CV settings. For VCV, only refer to the **preutterance section** of each explanation.

> H-row, y-row, and w-row have many examples, so they're explained in [14.Sound Types (Advanced)](/docs/tutorial-en-14-sound-types-advanced--docs).

## Fricatives

Fricative consonants strongly produce high frequencies (spectrogram shows darker colors at the top).
Consonants tend to be relatively long.

**Sounds using this pattern:**
- s-row, z-row
- sh-row, j-row (sha, ja)
- hy-row (hya)

**Key points:**

- **Left blank**: Must include the entire consonant, or it sounds like a different sound. While overly long consonants sound sluggish, prioritize not cutting the consonant start.

- **Overlap**: Set to about 1/3 of preutterance. For CV, overlap determines sound attack time. Making overlap too large makes the consonant head unclear, potentially sounding different, so don't raise it too much.

- **Preutterance**: The vowel begins where low frequencies start (where darker colors appear at the bottom of the spectrogram), so align preutterance here.

## Nasals

Nasal consonants characteristically produce weak sound in low frequencies and almost none in high frequencies (spectrogram shows light colors at the bottom, nearly colorless at the top).

**Sounds using this pattern:**
- n-row, m-row
- ny-row (nya), my-row (mya)

**Key points:**

- **Left blank**: Basically include the entire consonant, but nasals still sound the same with slightly cut consonant starts, so you can cut if the consonant is too long.

- **Overlap**: Set to about 1/3~1/2 of preutterance. Setting it slightly longer sounds smoother.

- **Preutterance**: As the vowel begins, darker colors also appear in the mid-range of the spectrogram, so set it here.

## Voiceless Plosives

Plosives consist of a section with strong sound across low to high frequencies (entire spectrogram showing dark colors), and **silent section just before**.
It's important to properly reflect the preparatory silent section in oto settings.

**Sounds using this pattern:**
- k-row, ky-row (kya)
- t-row, ty-row (tya)
- p-row, py-row (pya)
- ch-row (cha), ts-row (tsa)

> **Note:** Ch-row and ts-row show fricative-like spectrogram after silence (affricates), but the concept is the same - represent the silent section in settings.

**Key points:**

- **Left blank**: Set at where sound begins.

- **Overlap**: Set to about -20~-50. This negative overlap creates the preparatory section.

- **Preutterance**: As the vowel begins, high-frequency sounds weaken in the spectrogram, so set it here.

## Voiced Plosives

**Sounds using this pattern:**
- g-row, gy-row (gya)
- d-row, dy-row (dya)
- b-row, by-row (bya)
- r-row, ry-row (rya)

> **Note:** If g-row is nasalized, use same settings as nasals.

## Vowels

Vowels characteristically show dark colors in low-to-mid range.
Settings methods differ between CV and VCV.

**Sounds using this pattern:**
- a-row (a, i, u, e, o)
- n (moraic nasal)

If recorded correctly, "aa" should be pronounced as "a—" (if pronounced with a break like "a, a", better to re-record).
As long as all parameters fit within these 2 beats of "a", you don't need to be too particular about settings.

**Key points:**

Vowel sections all produce dark colors in low-to-mid range, plus they transition smoothly from previous to next sound, making settings somewhat difficult.

First identify that the spectrogram shows **[previous vowel] - [transition section] - [next vowel]**.
Place preutterance somewhere in this transition section, using playback buttons to find the right timing while listening.

The example above sets it where the transition ends and clearly becomes the next vowel.
