# Selecting a Folder

After loading your voicebank, choose which folder you want to edit and decide how to load its oto.ini.

## Choosing a Folder

From the dropdown menu (select box) at the top of the screen, select the folder you want to edit.

> 💡 **About Folders**: UTAU voicebanks are often organized by purpose or sound type  
> (Examples: by pitch like "C4", "A3", or by expression like "normal", "whisper")

### How to Select a Folder

1. **Open the Dropdown Menu**
   - Tap (or click) on the "Target Directory" dropdown
   - All folders in the voicebank will be displayed in a list

2. **Choose the Folder to Edit**
   - Tap the folder you want to edit from the list
   - The root folder (top level) is shown as "/"

3. **Choose How to Load Oto.ini**
   - After selecting a folder, tabs will appear at the bottom
   - Use the tabs to choose how to prepare the oto.ini

## How to Load Oto.ini

After selecting a folder, choose how to prepare the oto.ini.
Several methods are available depending on your situation:

### 📘 Basic (For Beginners)
Start here if you're new!

- **[06-1. Load from External Template](/docs/tutorial-en-06-1-load-from-template--docs)** - Use another oto.ini as a template (recommended for beginners)
- **[06-2. Use Existing Oto.ini](/docs/tutorial-en-06-2-load-existing--docs)** - When the folder already has an oto.ini (most common)
- **[06-3. Create New](/docs/tutorial-en-06-3-create-new--docs)** - When there's no oto.ini, create one with automatic settings

### 📗 Advanced (Once You're Comfortable)
For those who want more convenient methods.

- **[07-1. Continue from History](/docs/tutorial-en-07-1-load-from-history--docs)** - Resume from your previous editing session
- **[07-2. Multiple Aliases](/docs/tutorial-en-07-2-multiple-aliases--docs)** - Set multiple sounds in one wav (advanced automatic generation)

### 📕 Expert
For those who want fine-tuned control of settings.

- **[08. Advanced Customization](/docs/tutorial-en-08-advanced-auto-generation-settings--docs)** - Specify detailed parameters for automatic generation
