# Sound Types (Advanced)

This tutorial explains settings for h-row, y-row, and w-row.
These sounds have slightly different characteristics from other consonants.

> **Note:** This tutorial assumes CV settings. For VCV, only refer to the **preutterance section** of each explanation.

## H-row

H-row is a type of fricative but has different enough characteristics to be treated separately.
The consonant part shows weak sound spread across all frequencies (spectrogram looks like white noise).
Characterized by slow transition from consonant to vowel.

**Sounds using this pattern:**
- h-row (ha, he, ho)
- hy-row (hya), f-row (fa)

> **Note:** Hi and fu are explained separately later.

**Key points:**

- **Left blank**: Set at consonant start. Can cut if consonant is too long.

- **Overlap**: Set to about 1/3~1/2 of preutterance.

- **Preutterance**: Basically set at where fully transitioned to vowel. However, if transition is very slow by habit, setting in the middle of the transition section may be better - adjust while playing back.

## Y-row (Semi-vowel)

Y-row consonants have a spectrum similar to the vowel "i".
For example, "a ya" has a spectrum like **[a] - [a→i] - [i→a] - [a]**.

**Sounds using this pattern:**
- y-row (ya, yu, yo)

For example, "a wa" has a spectrum like **[a] - [a→u] - [u→a] - [a]**.

**Sounds using this pattern:**
