# Loading an Existing Oto.ini

If the folder contains an oto.ini file, you can load and edit it.
This is the most common usage.

## When to Use This Method

- ✅ **Want to modify an oto.ini you already created**
- ✅ **Want to adjust an oto.ini from a distributed voicebank**
- ✅ **Template included in the zip with wav files**

## Steps

### 1. Choose Loading Method

Select the "**Read Zip**" tab at the top of the dialog.

This tab is automatically selected if the folder contains an oto.ini.

    title="Read Zip tab selection screen"
  />
</div>

### 2. Verify Settings

Aliases (sound names) and configuration values from the file are displayed in a list.

In this preview screen:
- **Check for garbled text**: Verify aliases and filenames display correctly
- **Select character encoding**: Change encoding if text is garbled

> 💡 **Hint**: Oto.ini files are usually saved in Shift-JIS (standard for Japanese Windows),
> but if "あ" appears as "縺ゅ", try changing the encoding to "UTF-8" or another option.

### 3. Click "OK" to Start Editing

If everything looks good, tap "OK" to proceed to the editing screen.

## If Oto.ini is Not Found

If the folder doesn't contain an oto.ini, this tab cannot be used.
